## Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/renatohvo/js-antes-do-react.git

# baixar os pacotes do projeto
npm install

# executar o projeto
npm run dev

# url do projeto em execução
http://localhost:5173/

```
